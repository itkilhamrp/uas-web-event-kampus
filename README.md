# uas-web-event-kampus
# 
## Project ini merupakan form pendaftaran event kampus berbasis website
#
# Alur Algoritma Pengisian Form nya
1. User membukan website ini,
2. Lalu User Scroll ke bagian form,
3. User mengisi Form tersebut dari nama, email, no hp, dan pilih kategori,
4. Jika pengisian User nya benar dan lengkap sesuai ketentuan validasi javascript nya, maka akan muncul pesan sukses " Pendaftaran berhasil! Terima kasih telah mendaftar. ",
5. Namun, jika User salah mengisi maka akan muncul pesan error di setiap field nya dan muncul pesan error " Mohon lengkapi semua field dengan benar ",
6. Lalu, jika User berhasil submit, maka akan muncul di daftar list peserta,
7. Namun, jika pendaftaran telah penuh, karena kuota nya sudah mencapai maksimal 50/50, lalu User Submit (asalkan semua field nya telah di isi dengan benar dan lengkap), maka akan muncul pesan " Pendaftaran ditutup, kuota sudah penuh ",
8. Letak posisi elemen pesan nya di bawah tombol Submit/Daftar Sekarang.
